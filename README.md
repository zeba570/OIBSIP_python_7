# 🚀 **NEONCHAT – REAL-TIME MESSAGING PLATFORM**

## 🌟 **Overview**

NeonChat is a modern real-time messaging application built using **Node.js, Socket.IO, HTML, CSS, and JavaScript**. It provides instant communication through chat rooms with a stunning neon-themed user interface, smooth animations, and responsive design.
The application enables users to connect, chat, and collaborate in real time while enjoying a futuristic and interactive messaging experience.

# 🚀 **Installation**

### Clone Repository

```bash
git clone https://github.com/zeba570/OIBSIP_python_7.git
```
### Navigate to Project Folder

```bash
cd chat_application
```

### Install Dependencies

```bash
npm install
```

### Run Server

```bash
node backend/server.js
```

### Open in Browser

```bash
http://127.0.0.1:5500/index.html
```
